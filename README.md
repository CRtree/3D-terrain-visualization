# 3D-terrain-visualization  
这是一个基于openGL ES的三维地形可视化安卓程序。  
该程序实现的功能：  
（1）展示三维散点地形数据；  
（2）将三维散点图进行Delaunay三角剖分，生成三角网；  
（3）给三角网贴上地形纹理；  
（4）设置视角变换，加上光照。
